## 🎶 About the Project

**Lux Sundari** is an independent music project whose debut album *Nāda Potentia* (2023) explores the intersection of sacred sound, world mythology, and inner transformation. The album draws from Vedic, Greek, Egyptian and Buddhist traditions, weaving them into a progressive rock/metal sonic experience.

The name *Nāda Potentia* refers to the primordial power of sound — *Nāda* (Sanskrit: sound/vibration) and *Potentia* (Latin: power/potential).

## ✨ Features

- **Immersive dark aesthetic** inspired by progressive rock visuals
- **Lyrics modal** — click any track card to open full lyrics
- **EN / PT toggle** — switch between original and Portuguese translation inline
- **Spotify embed** — full album player integrated
- **Mobile responsive** — hamburger menu, adaptive layout
- **Scroll animations** — fade-up elements on scroll

## 🚀 Running locally

No build step required — pure HTML/CSS/JS.

```bash
# Clone the repo
git clone https://github.com/IgorLuna10/Lux-Sundari.git
cd Lux-Sundari

# Open directly in browser
open Lux_Sundari.html
```

Or use VS Code Live Server for hot reload.
